Local transcription with editorial clarity.

dictate.sh keeps speech workflows on-device. MLX delivers low-latency ASR, VAD finalizes turns, and a local LLM can extract intent when you need structure.

Features

Everything you need, nothing you don't.

MLX ASR

Fast, local transcription on Apple Silicon.

VAD turns

Finalize on silence, not arbitrary intervals.

Intent analysis

Optional on-device LLM for structure and routing.

Clean output

UI on stderr, stdout for scripting.

Usage

Configuration options.

$ uv run qwen3.py

$ uv run qwen3.py --analyze

$ uv run qwen3.py --model mlx-community/Qwen3-ASR-1.7B-8bit

$ uv run qwen3.py --vad-mode 0 --vad-silence-ms 1200 --min-words 6

Dial in the feel

Increase --vad-silence-ms to avoid short turns or lower --vad-mode for less aggressive detection.
